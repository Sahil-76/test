# Amazon Automation Assignment

## Description
Automated Amazon test cases using Playwright.

## Test Cases
1. Search iPhone → Add to cart → Print price
2. Search Samsung Galaxy → Add to cart → Print price

## Setup
npm install
npx playwright install

## Run
npx playwright test

## Parallel Execution
- fullyParallel: true
- workers: 2

## Tech Stack
- Playwright
- JavaScript
